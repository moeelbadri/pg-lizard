# pg-lizard



u need to install pgmetrics  (prefered globally /usr/local/bin)

https://pgmetrics.io/docs/install.html


wget https://github.com/rapidloop/pgmetrics/releases/download/v1.19.0/pgmetrics_1.19.0_linux_amd64.tar.gz
tar xvf pgmetrics_1.19.0_linux_amd64.tar.gz
cd pgmetrics_1.19.0_linux_amd64
./pgmetrics --help



https://pglizard.com/ then make account and create server and get the apiToken to use with script


once u do, add it to the .env
and
run the script
